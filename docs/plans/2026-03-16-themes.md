# Theme System Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Replace the binary dark/light toggle with a 6-theme picker (Dark, Light, Excel, Terminal, Nord, Pitch) living in the Group tab, purely cosmetic.

**Architecture:** Expand the existing CSS-variable theme system. Each theme is a `[data-theme="key"]` block in the `CSS` string. Excel mode additionally gets an `excel-mode` class on `AllPicksTable` for structural tweaks. The `dark` boolean state is replaced with a `theme` string. The nav toggle button is removed; theme selection moves to the Group tab.

**Tech Stack:** React (useState/useEffect), inline CSS variables, localStorage
